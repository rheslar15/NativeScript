The platform module allows you to get the available information for the device such as:
*  Device model
*  Device type
*  OS
*  OS version
*  Device sdkVersion
*  Device language
*  Device manufacturer
*  Device uuid

Also with its help, you could get information about the device screen:

* Screen scale
* Screen height DIPs
* Screen width DIPs
* Screen height Pixels
* Screen width Pixels
